## Steam

- Games are located on Windows in:

```
%CommonProgramFiles(x86)%\Steam\steamapps\common
```

- To open an URL inside the Steam Application on Windows:

Press ``` WinKey + R ``` and enter ``` steam://openurl/THE_URL ```.
